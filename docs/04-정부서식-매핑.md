## 시행지침 준수표 (프로젝트형, 시스템 반영 사항)

| 지침 규정 | 일움 반영 |
|---|---|
| 실행비 팀당 90만원×2개월=180만원 | 사업 규칙 기본값 `teamBudget` |
| 참여수당 1인 월 40만 / 멘토수당 월 15만 / 기업지원 월 5만 | 사업 규칙 기본값 (지급 관리 화면은 후속) |
| 실행비 절차: 수행계획서→품의→집행→결과서→잔액반납 | 문서 워크플로우 + 잔액 자동 계산 |
| 회의비: 식대 내부 1.5만/외부 3만, 다과 1만 (1인) | 회의록 작성 시 자동 검증 + 경고 |
| 주류 금지·22시 이후 결제 불가 | 회의록·PDF에 명문 표기, 안내 문구 |
| 교통비: 시내 팀당 일 2만, 자차 일 5만 | 회의록·출장보고서 검증 + 안내 |
| 강사비: 시간당 10만, 1일 3시간, 계좌이체(원천징수) | 강의보고서 입력 제한 + 안내 |
| 지원불가 항목(인건비·장비구매·주류 등) | 품의서 화면 경고 목록 |
| 비목별 증빙: 임차·재료·도서·인쇄→검수확인서 / 회의비→회의록 / 교통→출장보고서 / 강사비→강의보고서 | 지출 결과서 증빙 자동 매핑 (누락 시 제출 불가) |
| 팀 4인 내외·팀장 1명 | 팀 구성 구조 (팀장=등록자) |
| 수행계획서 개시 후 3일 내 제출 | 화면 안내 문구 |
| 실시보고 개시 후 5일 내 / 실적보고·결과보고 종료 후 10일 내 | 정부 보고 화면 안내 + 자동 생성 |
| 담당관 1인당 청년 50명 이하 | 보고 화면 초과 경고 |
| 서류 5년 보존 | DB 원본 보존 (삭제는 소프트 정책 후속 검토) |
