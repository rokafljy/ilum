# 일움 (Ilum)

**청년일경험사업 종합 운영관리 솔루션** — 고용노동부 청년일경험사업을 운영하는 전국 운영기관의 업무를 효율화하는 멀티테넌트 SaaS.

> 프로젝트명 "일움"은 내부 통칭입니다. (일 + 움트다 — 첫 일 경험이 움튼다)

## 배경

건국대 미래내일 일경험사업 포털 **영메이커스**(youngmakers.kr)를 실제 운영하며 검증한 업무 프로세스(팀 관리, 회의록 승인, 지출결과서·증빙, 통장대조, 일정, 멘토링)를 기반으로, 전국 ~300개 운영기관이 사용할 수 있는 대고객 솔루션으로 재구축한다.

## 비즈니스 모델

1. 운영기관 대상 솔루션 이용료 (연 단위 계약)
2. (장기) 익명화·통계 데이터 기반 부가서비스 — 벤치마크 리포트 등

## 로드맵

| Phase | 내용 | 상태 |
|---|---|---|
| 0 | 시장 검증 — 운영기관 인터뷰, 지불의사·요구 편차 확인 | 예정 |
| 1 | v2 기반 구축 — 멀티테넌트 DB, 인증, 핵심 워크플로우 | 예정 |
| 2 | 파일럿 — 건국대 마이그레이션 + 외부 2~3개 기관 | 예정 |
| 3 | 확장 — 영업, 셀프 온보딩, 통계 대시보드 | 예정 |
